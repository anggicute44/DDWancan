\# DDWancan



Aplikasi Berita Kampus merupakan platform informasi resmi yang dirancang untuk memudahkan mahasiswa dan staf dalam mengakses berita, pengumuman, dan kegiatan terbaru di lingkungan kampus. Aplikasi ini dikembangkan untuk menjawab kebutuhan akan media informasi yang terpusat, cepat, akurat, dan mudah diakses, sehingga seluruh civitas akademika dapat tetap terhubung dengan perkembangan kampus secara real-time.



Melalui sistem autentikasi berbasis akun kampusâ€”students untuk mahasiswa dan staff untuk dosen/stafâ€”aplikasi ini memastikan bahwa hanya pengguna terverifikasi yang dapat mengakses konten internal. Selain itu, staf tertentu diberi hak untuk mengunggah dan mengelola berita, sementara admin bertugas memverifikasi dan mempublikasikan informasi untuk menjaga kualitas dan keabsahan berita.



Aplikasi ini menyediakan fitur utama seperti pembacaan berita, kategori berita, pencarian cepat, notifikasi pengumuman penting, penyimpanan berita, serta sistem manajemen berita untuk staf dan admin. Dengan menggunakan arsitektur MVVM, database Room, dan pengembangan berbasis Kotlin dan Jetpack Compose, aplikasi ini dibuat untuk memberikan pengalaman pengguna yang cepat, responsif, dan konsisten.



Secara keseluruhan, aplikasi ini bertujuan menjadi pusat informasi kampus yang terpercaya, relevan, dan mudah digunakan, sekaligus meningkatkan efektivitas komunikasi internal antara mahasiswa, staf, dan admin kampus.
